O que são views em um framework web como o Flask?

  • Funções ou classes que lidam com solicitações HTTP e retornam respostas por meio das interações com as controllers.
  • Estruturas de dados usadas para armazenar informações de sessão do usuário.
  • Arquivos de estilo utilizados para estilizar páginas da web.
  • Componentes responsáveis por modelar os dados de um aplicativo.

Qual é o principal objetivo do framework Flask em Python?

  • Fornecer uma interface gráfica para a criação de aplicativos de desktop.
  • Facilitar a criação de aplicações web com Python.
  • Automatizar o processo de implantação de aplicativos em servidores.
  • Gerenciar o ciclo de vida de um aplicativo, desde o desenvolvimento até a produção.

Qual método HTTP é utilizado para solicitar recursos de um servidor web?

  • GET
  • POST
  • DELETE
  • PUT

O que é o protocolo HTTP?

  • Um protocolo de comunicação usado para transferir arquivos entre computadores.
  • Um protocolo de transferência de hipertexto usado para transmitir informações na web.
  • Um protocolo de criptografia usado para proteger dados confidenciais durante a transmissão.
  • Um protocolo de rede usado para criar redes locais em ambientes empresariais.

O que é uma rota em um aplicativo web Flask?

  • Um caminho físico que leva a um recurso no sistema de arquivos do servidor.
  • Um endereço de IP exclusivo atribuído a um servidor web.
  • Uma diretiva de configuração que define o comportamento de um aplicativo Flask.
  • Um URL específico que corresponde a uma função de view em um aplicativo Flask.
