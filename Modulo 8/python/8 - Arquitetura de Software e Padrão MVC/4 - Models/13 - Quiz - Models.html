O que é a pasta models em Python no contexto de desenvolvimento de software?

  • Um módulo que contém várias funções utilitárias.
  • Um módulo que representa um objeto de dados e define seu comportamento por meio das entidades e settings do banco de dados.
  • Uma função especial que manipula exceções de forma personalizada.
  • Um arquivo de configuração usado para armazenar informações de ambiente.

Qual é a função principal de um repositório em um sistema de controle de versão (VCS)?

  • Armazenar arquivos de backup para recuperação de desastres.
  • Facilitar a colaboração entre membros da equipe, rastreando mudanças e histórico de código.
  • Executar testes automatizados para garantir a qualidade do código.
  • Atuar como um servidor de banco de dados para armazenar dados relacionais.

O que são entidades em um contexto de desenvolvimento de banco de dados?

  • Funções que definem o comportamento de objetos de dados.
  • Módulos que encapsulam operações de entrada e saída.
  • Classes que representam tabelas de banco de dados e seus relacionamentos.
  • Variáveis globais que armazenam configurações de aplicativos.

Em Python, para que serve o bloco try-except?

  • Para criar loops iterativos sobre uma sequência de elementos.
  • Para definir métodos especiais de inicialização e encerramento de objetos.
  • Para capturar e lidar com exceções que ocorrem durante a execução do código.
  • Para definir blocos de código que serão executados em paralelo.

Qual é a finalidade do método especial `__enter__` em Python, usado em conjunto com a declaração `with`?

  • Para definir comportamentos personalizados quando um objeto é convertido em uma string.
  • Para manipular eventos de entrada do usuário em interfaces gráficas.
  • Para inicializar recursos antes de entrar em um bloco de código e limpar esses recursos após sair do bloco.
  • Para definir funções de comparação personalizadas entre objetos de uma classe.
